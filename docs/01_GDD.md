# 1. Présentation du projet

## Nom

**The Last Signal Online**

## Description

The Last Signal Online est un MMORPG de survie post-apocalyptique se déroulant dans un monde ouvert contaminé par une catastrophe scientifique.

Le joueur incarne l'un des derniers survivants immunisés face à une contamination ayant transformé une grande partie de l'humanité en créatures mutantes.

Le jeu met l'accent sur :

- l'exploration ;
- la coopération ;
- la survie ;
- les combats ;
- la progression du personnage ;
- la découverte de l'histoire du monde.

---

# 2. Vision du jeu

Notre objectif est de créer un MMORPG moderne proposant un monde persistant dans lequel les joueurs peuvent évoluer librement.

Le projet repose sur quatre piliers :

- un monde vivant ;
- une progression gratifiante ;
- une forte coopération entre joueurs ;
- une histoire évolutive racontée au fil des mises à jour.

Le jeu est pensé pour évoluer durant plusieurs années grâce à de nouveaux contenus.

---

# 3. Objectifs

Les objectifs du projet sont :

- proposer une aventure immersive ;
- offrir un monde ouvert riche à explorer ;
- permettre le jeu en solo comme en groupe ;
- construire une économie durable ;
- proposer des activités variées en dehors des combats.

---

# 4. Public visé

Le jeu s'adresse principalement :

- aux amateurs de MMORPG ;
- aux joueurs appréciant les jeux de survie ;
- aux joueurs coopératifs ;
- aux passionnés d'exploration ;
- aux joueurs appréciant la progression à long terme.

Âge recommandé :

**12 ans et plus**

# 6. Univers

Le monde est un vaste territoire ayant subi une catastrophe biologique mondiale.

Les anciennes villes sont désormais en ruines.

Les infrastructures militaires sont abandonnées.

La nature reprend progressivement ses droits.

Les créatures contaminées occupent désormais la majorité du territoire.

Le monde est entièrement connecté et persistant.

---

# 7. Scénario

Avant l'effondrement du monde, un scientifique extrêmement influent menait des recherches secrètes dans un laboratoire dissimulé.

Son objectif était de créer un humain génétiquement supérieur.

Les expériences réussirent…

Mais les sujets finirent par muter.

Ils s'échappèrent du laboratoire.

En quelques semaines, la contamination se propagea à travers le monde.

Le scientifique fut lui-même contaminé.

Avant de disparaître, il envoya un mystérieux message :

> **The Last Signal**

Aujourd'hui, les derniers survivants tentent de comprendre ce message.

Ils espèrent encore sauver l'humanité.

# Conclusion

The Last Signal Online est un projet ambitieux ayant pour objectif de proposer une expérience MMORPG moderne, évolutive et durable.

La priorité est de construire des fondations solides afin de permettre au jeu d'évoluer pendant plusieurs années sans nécessiter de refonte majeure.
