# 🚀 Objectif du projet

Créer **The Last Signal Online**, un MMORPG de survie post-apocalyptique proposant :

- 🌍 Un monde persistant
- 👥 Plusieurs centaines de joueurs simultanés
- ⚔️ PvE et PvP
- 🏰 Guildes
- 💰 Économie entre joueurs
- 🛠 Artisanat
- 📖 Une histoire évolutive
