# 🚀 Objectif du projet

Créer **The Last Signal Online**, un MMORPG de survie post-apocalyptique offrant un monde persistant, un gameplay coopératif et compétitif, ainsi qu'une architecture moderne capable d'accueillir plusieurs centaines de joueurs simultanément.
