# 1. Présentation

L'univers de **The Last Signal Online** est un monde post-apocalyptique où la civilisation s'est effondrée après une catastrophe scientifique.

Les infrastructures sont en ruines, la nature reprend ses droits et des créatures contaminées parcourent désormais les terres.

Les survivants tentent de reconstruire leur vie tout en cherchant des réponses sur l'origine de la catastrophe.

---

# 2. Contexte

Avant la catastrophe, l'humanité connaissait une période de grands progrès scientifiques.

Dans le plus grand secret, un scientifique dirigeait un laboratoire où étaient menées des expériences sur l'amélioration génétique de l'être humain.

Ces recherches provoquèrent une contamination mondiale.

En quelques semaines, la société s'effondra.

---

# 3. Le monde

Le monde est entièrement ouvert.

Les joueurs peuvent explorer librement les différentes régions.

Chaque zone possède :

- son climat ;
- ses ressources ;
- ses dangers ;
- ses créatures ;
- son histoire.

Certaines régions seront plus dangereuses que d'autres.

---

# 4. Les régions

Le monde sera composé de nombreuses zones.

Exemples :

- villes abandonnées ;
- villages détruits ;
- forêts contaminées ;
- montagnes ;
- laboratoires secrets ;
- bases militaires ;
- usines ;
- marais ;
- grottes ;
- bunker souterrains.

Chaque région possède sa propre identité visuelle.

---

# 5. La contamination

La contamination est l'élément central du monde.

Elle transforme progressivement les êtres vivants.

Certaines zones sont fortement contaminées.

D'autres restent relativement sûres.

Plus la contamination est importante, plus les ennemis deviennent puissants.
---

# 5.1 Niveaux de contamination

Le monde est divisé en plusieurs niveaux de contamination.

Chaque niveau influence :

- les créatures présentes ;
- les ressources disponibles ;
- la difficulté ;
- les récompenses ;
- les risques pour les joueurs.

## 🟢 Zone Verte

Les zones vertes sont les plus sûres.

Elles accueillent les principaux camps de survivants ainsi que les nouveaux joueurs.

Les créatures y sont peu nombreuses et relativement faibles.

---

## 🟡 Zone Jaune

Les zones jaunes représentent les premières régions contaminées.

Les joueurs y rencontrent leurs premiers véritables défis.

Les ressources y sont plus abondantes mais les dangers augmentent.

---

## 🔴 Zone Rouge

Les zones rouges sont fortement contaminées.

Les créatures y sont nombreuses et beaucoup plus agressives.

Les joueurs devront être correctement équipés avant de s'y aventurer.

---

## ⚫ Zone Noire

Les zones noires représentent le cœur de la contamination.

Ce sont les endroits les plus dangereux du monde.

On y trouve :

- les créatures les plus puissantes ;
- les boss majeurs ;
- des laboratoires oubliés ;
- des technologies perdues.

La Montagne du Dernier Signal se situe dans une Zone Noire.

---

# 5.2 Les Signaux

Le phénomène appelé **"The Last Signal"** est au cœur de l'univers.

À certains endroits du monde, d'étranges émissions radio peuvent être détectées.

Ces signaux semblent avoir une influence inconnue sur les créatures contaminées.

Leur origine demeure un mystère.

Les joueurs découvriront progressivement leur véritable nature au cours de l'histoire principale.

---

# 5.3 La Montagne du Dernier Signal

Au centre du continent se dresse une immense montagne visible depuis de nombreuses régions.

Au sommet se trouve une gigantesque cascade derrière laquelle est cachée une ancienne base scientifique.

Cette montagne constitue le principal objectif de l'aventure.

Elle est considérée comme l'endroit le plus dangereux du monde.

Les survivants la surnomment :

> **La Montagne du Dernier Signal**

---

# 5.4 Les villes

Avant la catastrophe, plusieurs grandes villes occupaient le territoire.

Aujourd'hui, elles sont en ruines.

Chaque ville possède :

- sa propre histoire ;
- son architecture ;
- ses habitants avant la catastrophe ;
- ses ressources ;
- ses secrets.

Certaines villes pourront être partiellement sécurisées par les joueurs au fil du développement.

---

# 5.5 Les anomalies

La catastrophe a laissé derrière elle d'étranges phénomènes.

Parmi eux :

- des tempêtes électromagnétiques ;
- des zones où les communications deviennent impossibles ;
- des perturbations lumineuses ;
- des anomalies gravitationnelles ;
- des champs de contamination instables.

Ces anomalies pourront modifier temporairement le gameplay et créer des événements imprévisibles.

---

# 6. Les survivants

Les joueurs incarnent des survivants immunisés.

Ils représentent l'un des derniers espoirs de l'humanité.

Ils peuvent :

- explorer ;
- commercer ;
- combattre ;
- coopérer ;
- fonder des guildes ;
- reconstruire certaines zones.

---

# 7. Les créatures

Les créatures sont les principales menaces.

Elles proviennent :

- d'humains contaminés ;
- d'animaux mutés ;
- d'expériences scientifiques.

Chaque famille de créatures possède :

- un comportement ;
- des capacités ;
- des forces ;
- des faiblesses.

---

# 8. Les ressources

Les joueurs peuvent récolter :

- bois ;
- pierre ;
- métaux ;
- nourriture ;
- composants électroniques ;
- carburant ;
- médicaments.

Ces ressources permettent :

- l'artisanat ;
- les réparations ;
- les constructions ;
- le commerce.

---

# 9. Les mystères

Le monde cache de nombreux secrets.

Parmi eux :

- le laboratoire principal ;
- les recherches du scientifique ;
- les bunkers oubliés ;
- des documents confidentiels ;
- l'origine du "Last Signal".

Les joueurs découvriront progressivement ces éléments.

---

# 10. Ambiance

L'univers doit transmettre plusieurs émotions :

- solitude ;
- tension ;
- exploration ;
- espoir ;
- mystère.

Le monde raconte une histoire à travers ses décors, ses bâtiments et les traces laissées par l'humanité.
