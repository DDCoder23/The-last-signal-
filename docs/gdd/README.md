# 🎮 Game Design Document (GDD)

> Documentation de conception de **The Last Signal Online**

Le **Game Design Document (GDD)** décrit l'ensemble des mécaniques, de l'univers et des systèmes de jeu de **The Last Signal Online**.

Il constitue la référence principale pour tous les membres de l'équipe.
