# 🎮 Game Design Document (GDD)

> Documentation de conception du jeu **The Last Signal Online**

Le **Game Design Document (GDD)** est la référence principale pour la conception du jeu. Il décrit l'univers, les mécaniques de gameplay, les systèmes de progression, les fonctionnalités et la vision globale du projet.

Tous les membres de l'équipe doivent s'appuyer sur cette documentation lors du développement.
